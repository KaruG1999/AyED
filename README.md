# Algoritmos y Estructuras de Datos

Materia central en la formación en informática enfocada en el estudio de **estructuras de datos avanzadas y análisis de algoritmos**.

El objetivo principal es aprender a **modelar problemas complejos y resolverlos de forma eficiente**, utilizando las estructuras adecuadas y evaluando el costo computacional de cada solución.

Este repositorio contiene:

- apuntes teóricos
- implementaciones en Java
- resolución de trabajos prácticos
- ejercicios de análisis de complejidad
- material de preparación para evaluaciones

---

## Objetivo de la materia

Desarrollar la capacidad de:

- elegir la estructura de datos adecuada para cada problema
- analizar la eficiencia de algoritmos
- implementar estructuras complejas correctamente
- comprender el impacto del diseño en el rendimiento

---

## Conceptos centrales

### Recursión

Muchos problemas complejos pueden definirse de forma recursiva.

Se estudia:

- definición de problemas recursivos
- implementación en estructuras dinámicas
- relación con estructuras como árboles

---

### Árboles

Estructuras jerárquicas utilizadas para representar información organizada.

Se estudian:

- árboles binarios
- árboles generales
- recorridos (inorden, preorden, postorden)
- aplicaciones como árboles de expresión

---

### Heap y colas de prioridad

Estructuras diseñadas para mantener elementos ordenados según prioridad.

Aplicaciones típicas:

- selección eficiente
- algoritmos de ordenamiento
- planificación de tareas

---

### Grafos

Modelos que permiten representar relaciones complejas entre elementos.

Se estudian:

- grafos dirigidos y no dirigidos
- matrices y listas de adyacencia
- algoritmos de recorrido (DFS y BFS)
- componentes conexas
- caminos mínimos

---

### Análisis de algoritmos

Estudio formal de la eficiencia de los algoritmos.

Se trabaja con:

- notación Big-O
- caso promedio, peor caso y mejor caso
- comparación entre estrategias algorítmicas

---

## Qué se aprende realmente en esta materia

Esta materia desarrolla habilidades fundamentales para cualquier área de la informática:

- pensamiento algorítmico
- optimización de soluciones
- modelado de problemas complejos
- análisis de eficiencia

Muchas técnicas vistas aquí aparecen luego en:

- motores de búsqueda
- inteligencia artificial
- sistemas distribuidos
- optimización de software
